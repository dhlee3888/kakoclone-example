CSS Selectors and Pseudo Selectors

Selector 사용 유형

[속성 선택자] h1[title] {color:red;} 태그/속성명 일치시 h1[title="apple"] {color:red;} 태그/속성명/속성값 일치시 [title] { } 속성명 일치 [title="value"] 속성명/속성값 일치시 속성 선택자에서는 [ ], ~=, *=, |=, ^=, $= 등의 기호를 사용합니다.
[순서 order] :nth-child(2n+1) (odd) (even) (-n+3) { } :first-clild { } :last-child { } 동일한 tag 가 child 에 있을때만 적용됨
